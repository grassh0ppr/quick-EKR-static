About Me

Hey, I'm Emma! I am a web developer + UI/UX designer with a music & arts background, and a soft spot for solving tricky problems with style. I got into web development during the pandemic, teaching myself the ropes as the world went on pause. What started as a curious detour turned into a full-blown infatuation with building clean, accessible, and harmonious digital experiences for everyone.

My dev stack includes HTML, CSS, JavaScript, Node, React, and even some ASP.NET, but I'm just as happy in Figma or Illustrator dreaming up visuals. I also enjoy making art (fun fact: the background image of this page is my watercolor), and I love finding ways to blend that creative energy into the work I do—whether it's crafting a tidy, formal interface for a government website or cranking out bold, dynamic layouts that still have just the right Feng Shui.

I'm always chasing that sweet spot where logic meets art. And when I'm not working, I'm probably reading something, walking the trail, sketching on my iPad, or helping friends untangle their AI sloppy vibe-coded websites. üòâ

Recent Work

Check out my GitHub for more projects.

Group Benefits Website

I was assigned this project as a front-end developer at a third-party admin company that handles the State of New Mexico Group Benefits Plan in 2022. I was responsible for the design and development of the website, which included creating a custom brand theme, implementing responsive design, and optimizing the site for performance. I also refactored the legacy code to meet WCAG 2.1 compliance. This is pretty much an on-going project; the site has evolved quite a bit since the HCA merger in June 2024.

Release of Health Info Authorization Form

I created an HTML form where users can securely submit an authorization for the release of their health information. Users can also upload any supporting documents. The form generates a PDF from the user's data and sends the form data to the back end, where it is forwarded in our email system to the appropriate contact points at the PSIA. The form is WCAG compliant and mobile responsive, utilized regularly by our users, and has been a great addition to the resources available on the website.

NMPSIA District Contacts

I created a "database" interface of all the district contacts for the NMPSIA, retrieved from our district contact API via fetch request. The database is searchable, and it includes contact information for various staff role categories for each district. The database is updated regularly and is frequently used by our client to communicate with the districts.

Book Club React App

I am working on a book club app using React JS. The app allows users to create a book club, invite members, and manage their reading list. The app is fully responsive and includes a calendar feature for scheduling meetings. I used React Router for navigation and Axios for API calls.

Pack N Go React App

I am working on a packing list app using React JS. The app allows users to create a packing list for their trips, and it will have a feature for sharing the list with others. A work in progress.

To-Do List React App

I created a simple to-do list app using React JS. The app allows users to add, edit, and delete tasks. It also includes a feature for marking tasks as completed. The app is fully responsive and uses local storage to save the tasks.
